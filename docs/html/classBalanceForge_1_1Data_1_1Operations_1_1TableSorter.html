Подробное описание

Оптимизированный сортировщик строк баланса с поддержкой различных типов данных. Использует Array.Sort для высокой производительности вместо LINQ OrderBy.

Сортирует список строк баланса по указанному столбцу в заданном направлении. Операция оптимизирована для работы с большими наборами данных благодаря использованию Array.Sort.

Аргументы
rowsСписок строк баланса для сортировки.
columnIdИдентификатор столбца для сортировки.
directionНаправление сортировки (Ascending, Descending или None).
columnTypeТип данных столбца для правильного сравнения значений.
Возвращает
Отсортированный список строк. Если direction равен None или список содержит 1 или менее элементов, возвращается исходный список.
