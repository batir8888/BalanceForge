Подробное описание

Интерфейс для импортеров данных из различных форматов файлов в таблицы баланса. Позволяет расширять функциональность добавлением новых импортеров (JSON, XML, Excel и т.д.).

bool BalanceForge.ImportExport.IImporter.CanImport ( string filePath)

Проверяет может ли импортер обработать файл по его расширению или другим критериям.

Аргументы
filePathПуть к файлу для проверки поддержки.
Возвращает
true если импортер может обработать этот файл, иначе false.
