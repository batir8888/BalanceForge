Подробное описание

Статический сервис управления буфером обмена для таблицы баланса. Поддерживает копирование и вставку отдельных значений и групп значений. Интегрирует внутренний буфер обмена с системным буфером обмена Unity через GUIUtility.systemCopyBuffer.
