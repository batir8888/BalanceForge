Подробное описание

Представляет строку таблицы баланса с поддержкой различных типов данных и Unity Object ссылок. Обеспечивает сериализацию и десериализацию значений ячеек с сохранением информации о типах. Поддерживает примитивные типы (int, float, bool), Vector2/3, Color и Asset References.
