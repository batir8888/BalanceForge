Подробное описание

Определяет схему столбца таблицы баланса, включая тип данных, валидацию и значение по умолчанию. Поддерживает сериализацию/десериализацию различных типов (примитивы, Vector, Color, Enum, Asset References). Может содержать кастомный валидатор для проверки значений.

◆ SetAssetType()

Устанавливает тип Unity Object для Asset Reference столбцов. Сохраняет assembly-qualified имя типа для восстановления при десериализации.

Аргументы

◆ Validate()

bool BalanceForge.Core.Data.ColumnDefinition.Validate ( object value)

Проверяет соответствие значения правилам валидации столбца. Если столбец обязательный, проверяет что значение не null и не пусто. Если установлен кастомный валидатор, также выполняет его проверку.

Аргументы
valueЗначение для проверки.
Возвращает
true если значение проходит всю валидацию, иначе false.
