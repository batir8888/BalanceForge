Элементы перечислений
String 

Текстовая строка.

Integer 

Целое число (int).

Float 

Число с плавающей точкой (float).

Boolean 

Логическое значение (bool) - true или false.

Enum 

Перечисление - выбор из предопределенного списка значений.

AssetReference 

Ссылка на Unity Asset (Prefab, Material, Texture2D и т.д.).

Color 

Цвет в формате RGBA (Color).

Vector2 

Двумерный вектор (Vector2) с компонентами x и y.

Vector3 

Трехмерный вектор (Vector3) с компонентами x, y и z.

◆ ErrorSeverity

enum BalanceForge.Core.Data.ErrorSeverity

Перечисление уровней серьезности ошибок валидации. Определяет как критичной является найденная проблема.

Элементы перечислений
Warning 

Предупреждение - потенциальная проблема, но данные могут быть использованы.

Error 

Ошибка - нарушение требований, данные не должны использоваться в продакшене.

Critical 

Критическая ошибка - серьезное нарушение целостности данных, использование невозможно.
